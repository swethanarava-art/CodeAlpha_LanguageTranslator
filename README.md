# 🌍 AI Language Translator

## 📌 Overview

The AI Language Translator is a Python-based web application that translates text between multiple languages using an AI-powered translation library. It provides a simple and interactive interface built with Streamlit.

## 🚀 Features

* Translate text between multiple languages
* User-friendly Streamlit interface
* Fast and accurate translations
* Supports various source and target languages

## 🛠️ Technologies Used

* Python
* Streamlit
* Googletrans

## 📂 Project Structure

```
CodeAlpha_LanguageTranslator/
│── app.py
│── requirements.txt
│── README.md
```

## ▶️ Installation

1. Clone the repository
2. Install dependencies

```
pip install -r requirements.txt
```

3. Run the application

```
streamlit run app.py
```

## 🎯 Output

* Users can enter text in one language.
* The application translates it into the selected target language.
